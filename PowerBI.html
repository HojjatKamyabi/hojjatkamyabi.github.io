Car Sales Dashboard - Power BI

View Project on GitHub

Project Overview

This Car Sales Dashboard provides an interactive and data-driven analysis of car sales in Germany. It offers insights into sales trends, average prices, mileage, fuel types, and the most popular car models. The dashboard allows users to filter data by brand, fuel type, and year, making it a useful tool for understanding market trends and supporting data-driven decision making in the automotive sector.

Dataset

Source: AutoScout24 Germany, a popular car marketplace

Description: The dataset includes comprehensive information about vehicle sales:

  • Car brands and models
  • Manufacturing year
  • Mileage information
  • Fuel type (Gasoline, Diesel, Electric/Gasoline)
  • Pricing data
  • Sales volume

Key Performance Indicators

  • Total Average Price: Displays the average price of all cars in the dataset
  • Total Sales Number: Shows the total number of cars sold
  • Most Popular Car: Identifies the car model with the highest number of sales
  • Average Mileage: Presents the average mileage across all vehicles
  • Average Vehicle Age: Calculates the average age of vehicles in the dataset

Dashboard Components

1 Interactive Filters

  • Brand Selector: Allows users to filter sales by a specific car brand
  • Fuel Type Filter: Segments the data based on fuel type (Gasoline, Diesel, Electric/Gasoline)
  • Year Range Slider: Enables users to analyze sales data within a specific year range
  • Price Range Filter: Lets users focus on specific price segments

2 Data Visualizations

  • Top 10 Models - Average Year: Line chart showing the average manufacturing year of the top 10 car models
  • Top 10 Models - Average Mileage: Line chart analyzing the average mileage of the top 10 car models
  • Fuel Type Distribution: Pie chart illustrating the share of different fuel types in the dataset
  • Price Trend Analysis: Area chart showing pricing trends over time

3 Summary Tables

  • Top 10 Car Models Summary: Table displaying key statistics including price average, number of sales, and total revenue per model
  • Brand Performance Matrix: Comparative analysis of different brands by sales volume and average price
  • Year-over-Year Comparison: Summary of sales performance across different years

Technical Implementation

  • Data Modeling: Created relationships between different data tables for comprehensive analysis
  • DAX Measures: Developed custom DAX formulas to calculate KPIs and metrics
  • Advanced Visuals: Implemented custom visuals for enhanced data representation
  • Slicers & Filters: Configured interactive filtering capabilities for user-driven analysis
  • Data Transformation: Applied Power Query for data cleaning and preparation

How to Use the Dashboard

To explore this interactive dashboard on your local machine:

1. Download the Power BI file:

Download the .PBIX file from GitHub: https://github.com/HojjatKamyabi/PowerBI_Car_Sale

2. Open with Power BI Desktop:

Open the file using Microsoft Power BI Desktop (free download from Microsoft)

3. Using the dashboard:

  • Use the interactive filters on the right panel to segment the data by your criteria of interest
  • Hover over charts for detailed tooltips with additional information
  • Click on segments in charts to cross-filter other visualizations
  • Use the navigation buttons at the top to switch between different views
  • Export insights as images or PDF reports using Power BI's export functionality

Tools & Technologies

  • Microsoft Power BI: Primary development platform
  • DAX (Data Analysis Expressions): For creating measures and calculated columns
  • Power Query: For data transformation and preparation
  • Excel: Source data storage format
  • Custom Visuals: Enhanced visualization components
