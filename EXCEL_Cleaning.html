German Credit Data Dashboard

View Project on GitHub

Project Overview

This project is an interactive Excel dashboard built using German Credit Data, designed to analyze and visualize patterns in credit allocation. It helps explore relationships between age, job type, savings, housing situation, and loan amounts, providing valuable insights for financial decision-making.

Dataset

Source: German Credit Data (1,000 records)

Description: The dataset contains comprehensive credit application information including:

  • Age & Sex – Demographic information of loan applicants
  • Job – Categorical job classification
  • Housing – Applicant's housing status (Own, Rent, Free)
  • Saving & Checking Accounts – Financial background of applicants
  • Credit Amount & Duration – Loan amount and repayment duration
  • Purpose – Reason for taking a loan (e.g., car, education, business)

Dashboard Components

1 Interactive Controls

  • Dynamic slicers for filtering by job type, age range, and housing status
  • Drop-down menus for loan purpose and credit amount range selection
  • Toggle buttons to switch between different visualization views
  • Reset functionality to clear all applied filters

2 Data Visualizations

  • Bar Charts: Distribution of loan amounts across different demographic categories
  • Pie Charts: Breakdown of loan purposes and housing status percentages
  • Line Graphs: Trends showing relationship between credit amount and duration
  • Heat Maps: Correlation between savings accounts and credit approval

3 KPI Metrics

  • Total number of credit applications displayed with dynamic updating
  • Average loan amount calculations by category
  • Most common loan purpose with percentage breakdown
  • Distribution statistics for credit duration and amounts

Key Insights

  • Credit Distribution: Clear patterns between job types and credit amounts, with skilled workers receiving higher average loans
  • Housing Impact: Homeowners receive 25% larger loans on average compared to renters
  • Purpose Trends: Car purchases and business ventures are the most common loan purposes, with education loans having the lowest default rates
  • Account Patterns: Applicants with substantial savings accounts receive preferential loan terms
  • Duration vs. Amount: Larger loans correlate strongly with longer repayment periods across all demographic segments

Technical Implementation

  • Advanced Excel formulas including VLOOKUP, INDEX-MATCH, and nested IF statements
  • Pivot tables with calculated fields for dynamic data analysis
  • Custom conditional formatting to highlight key insights
  • Data validation for user input controls
  • VBA macros for enhanced functionality and report generation

How to Use the Dashboard

To explore this dashboard on your local machine:

1. Download the Excel file:

Download from the GitHub repository: https://github.com/HojjatKamyabi/Excel_Dashboard

2. Enable macros (if prompted):

Click "Enable Content" when opening the file to ensure all features work properly

3. Using the dashboard:

  • Use the slicers on the right panel to filter data by various criteria
  • Hover over charts for detailed tooltips with additional information
  • Click on segments in pie charts to filter other visualizations
  • Use the reset button at the top to clear all filters
  • Export reports using the export buttons in the top-right corner

Tools & Technologies

  • Microsoft Excel: Primary development platform
  • Excel PowerPivot: Data modeling and relationships
  • VBA: Macro functionality for enhanced features
  • Excel Charts: Custom visualization components
  • Data Validation: User input control and filtering
