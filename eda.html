Customer Churn Analysis

View Project on GitHub

Project Overview

This project explores customer churn behavior through comprehensive analysis of a telecom dataset. By applying data science techniques and statistical methods, I identified key patterns and factors that influence customer decisions to leave the service. The ultimate goal was to extract actionable insights that can drive business decisions and develop retention strategies.

Dataset

Source: Telecom customer dataset (CSV format)

Description: The dataset contains detailed customer information including:

  • Demographic details (gender, age, partner status)
  • Service subscription information (phone, internet, streaming services)
  • Account information (tenure, contract type)
  • Billing details (payment method, monthly charges, total charges)
  • Churn label indicating whether a customer left the service

Analysis Methodology

1 Data Cleaning

  • Audited dataset for missing values (confirmed none present)
  • Converted categorical variables using One-Hot Encoding for analysis compatibility
  • Applied log transformation to address skewness in numerical features
  • Standardized text fields and removed inconsistencies

2 Exploratory Data Analysis (EDA)

  • Univariate Analysis: Examined distribution patterns of key variables (MonthlyCharges, Tenure, etc.)
  • Bivariate Analysis: Investigated relationships between individual features and churn
  • Multivariate Analysis: Created correlation heatmaps to identify feature interactions
  • Generated statistical summaries and visualizations to highlight patterns

3 Feature Engineering

  • Created TotalMonthlySpend (MonthlyCharges × Tenure) to assess lifetime value
  • Derived AutoPay feature from PaymentMethod to analyze payment behavior
  • Developed customer segments based on contract types and service subscriptions
  • Generated tenure-based cohorts to analyze churn patterns over time

Key Insights

  • Price Sensitivity: Customers with higher Monthly Charges have significantly increased churn probability (1.7x higher for top quartile vs bottom quartile)
  • New Customer Vulnerability: Customers with less than 12 months tenure churn at a rate 3x higher than long-term customers
  • Payment Method Impact: Customers using AutoPay options demonstrate 45% lower churn rates
  • Contract Effect: Month-to-month contracts show 5x higher churn rate compared to two-year contracts
  • Bi-Modal Distribution: Churn occurs at both extremes of the spending spectrum, indicating different issues for each segment

Business Recommendations

  • Implement targeted retention offers for customers approaching the 12-month mark
  • Create incentives for customers to switch from month-to-month to longer-term contracts
  • Develop AutoPay adoption campaigns highlighting convenience and potential discounts
  • Address high-value customer pain points with personalized service improvements
  • Introduce loyalty rewards based on tenure milestones to encourage retention

How to Run the Code

To run this analysis on your local machine:

1. Install required libraries:

pip install pandas numpy matplotlib seaborn ydata-profiling

2. Clone the repository:

git clone https://github.com/HojjatKamyabi/EDA_TelCo.git cd EDA_TelCo

3. Run the Jupyter notebook or Python script:

  • Load the dataset
  • Perform Data Cleaning & Preprocessing
  • Execute EDA and Feature Engineering
  • Analyze insights and interpret results
  • Visualizations and results will be displayed accordingly

Technologies Used

  • Python: Primary programming language
  • Pandas & NumPy: Data manipulation and analysis
  • Matplotlib & Seaborn: Data visualization
  • Jupyter Notebook: Interactive development environment
  • YData-Profiling: Automated EDA reporting
